Chrome 확장 프로그램 설치

1단계: 텍스트를 강조 표시하고 질문하세요

질문하거나 번역하려는 텍스트를 강조 표시한 후, “Ask AI” 또는 “Translate” 버튼을 클릭하세요. 탭을 전환할 필요가 없습니다.

2단계: 팝업이 즉시 표시됩니다

선택한 AI(ChatGPT, Claude, Gemini 등)가 스마트 팝업으로 즉시 응답합니다. 복사, 편집, 계속 대화도 가능합니다.

3단계: 보안 유지

QuickAsk는 AI에 보내기 전에 토큰, 이메일, URL 등 민감한 정보를 자동으로 마스킹합니다.
